API reference

The Comic Discovery API runs on localhost:3000. Built with Node.js, Express, PostgreSQL, and Prisma ORM.

Data model

Characters connect through issues via IssueCharacter. Each link carries a role: main, supporting, villain, or guest. Issues carry isKeyIssue, isFirstAppearance, isMajorCrossover.
